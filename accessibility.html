Accessibility Statement

This website aims to be readable, responsive, and usable across devices.

Accessibility goals

The site uses semantic HTML, readable typography, responsive layouts, keyboard-accessible controls, alt text for meaningful images, and visible focus states.

Known limits

Some third-party embeds, images, or external services may have separate accessibility behavior. The site owner should review accessibility after every major content update.

Feedback

If you find an accessibility issue, contact the site owner with the page URL and a description of the problem.
